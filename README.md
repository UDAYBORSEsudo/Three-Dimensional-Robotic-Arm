# Three-Dimensional-Robotic-Arm
Three-Dimensional Robotic Arm 🤖  A 3-DOF 3D-printed robotic arm controlled using real-time hand gestures. The system uses Python, OpenCV, and MediaPipe for hand tracking and sends control commands to an Arduino Uno through serial communication. Servo motors control the base, arm movement, and gripper for precise robotic manipulation.
